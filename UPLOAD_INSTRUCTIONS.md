# Upload Instructions

Upload these files into your existing GitHub repository:

https://github.com/SilverNightFall/youtube-healthmanifested-playlist-songs

Best option:

1. Download this zip.
2. Unzip it on your computer.
3. In GitHub, click Add file, then Upload files.
4. Drag in the unzipped files and folders.
5. Use this commit message: Add playlist index and structured AI metadata
6. Click Commit changes.

If GitHub will not accept folders in the browser, upload the root files first:

- playlist-index.json
- playlist-index.md
- playlist-index.schema.json
- AI_USAGE.md

Then create folders manually using Add file, Create new file, and type a path like:

metadata/human-design/human_design_gate_line_songs.json

GitHub creates the folder automatically when you type the slash in the filename.
